# Feature Kind

## Kinds

### Feature

- kind: kind_definition
- name: feature
- ancestor: entity
- attributes:
  - source: string
  - usage: string
  - effect: string
  - mechanics_module: string

> A class feature, racial trait, background benefit, or passive monster ability.
> Authored once in the shared catalog (or inline on a character/monster as an
> H5 sub-entity override). Characters reference features by id in their
> features list.
