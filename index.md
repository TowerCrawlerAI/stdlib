# TowerAI Standard Library

[core](core/index.md)

[dnd5e](dnd5e/index.md)

- name: stdlib
- version: 0.1.0

> Canonical entry point for the TowerAI standard library. Imports the
> system-agnostic core layer first (world model, game object kinds, verbs,
> events), then the default rule-system layer (currently D&D 5e). Floor and
> party authors should import this file by convention: `[stdlib](stdlib)`.
